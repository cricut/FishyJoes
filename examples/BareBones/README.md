# BareBones Example for FishyJoes

This is a package containing the files needed to start building a cross-platform library with FishyJoes.

It has already had `fishy-joes package-init` generate boiler-plate files and includes minimal tests for each supported foreign language.

It does not yet include bindings, which can be generated and tested by executing:

`swift run fishy-joes generate build test --wasm --kotlin-fast --c-sharp --dart`
